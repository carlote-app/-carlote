# 🤖 Carlote

> QA com IA para desenvolvedores brasileiros

A Carlote é uma IA de Quality Assurance que analisa seu código automaticamente, detecta bugs, gera casos de teste e produz relatórios profissionais de QA — em português, com preço em reais.

---

## ✅ O que a Carlote faz

- 🔍 Detecta bugs por funcionalidade
- 📋 Gera casos de teste (CT-001, CT-002...)
- 🐛 Relatório de bugs (BUG-001, BUG-002...)
- 📄 Salva relatório automaticamente
- 🇧🇷 100% em português brasileiro

---

## 🚀 Como usar
```bash
# Instalar
npm install

# Analisar um arquivo
node src/carlote.js seuarquivo.js
```

## 🛠️ Stack

- Node.js
- Groq API (Llama 3.3)
